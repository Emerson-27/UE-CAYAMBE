VISIÓN, MISIÓN Y OBJETIVO

VISIÓN

Ser un referente en viviendas inteligentes sostenibles, liderando la transformación hacia un estilo de vida más consciente, eficiente y accesible, donde la innovación tecnológica y la sostenibilidad se conviertan en la norma para hogares del futuro.

MISIÓN

Proporcionar hogares que integren tecnología avanzada y energías renovables, facilitando el día a día de sus habitantes con automatización eficiente y fomentando prácticas responsables con el medio ambiente, siempre con un enfoque inclusivo y adaptable a las necesidades de cada usuario.

OBJETIVOS GENERALES Y ESPECIFICOS

OBJETIVO GENERAL

Diseñar e implementar una casa domótica que utilice energía solar como fuente principal, mejorando la eficiencia energética y ofreciendo soluciones automatizadas que incrementen la comodidad, seguridad y sostenibilidad, adaptándose a las necesidades de cualquier usuario.

OBJETIVO ESPECIFICO

  1. Incorporar sistemas automatizados para controlar iluminación, seguridad y otros aspectos del hogar de manera eficiente.
  2. Optimizar el consumo energético a través de la instalación de paneles solares, promoviendo el uso de energías renovables.
  3. Diseñar interfaces accesibles y fáciles de usar para garantizar que personas de todas las edades y capacidades puedan interactuar con los sistemas domóticos.
  4. Reducir el impacto ambiental del hogar mediante la integración de tecnologías sostenibles y prácticas responsables.

DESAROLLO DEL PROYECTO

  • MATERIALES

Los materiales que se utilizo para la construccion de la casa domotica es:

  1. Una tabla triplex
  2. Laminas de carton
  3. Temperas de colores
  4. Laminas de imagenes y papel celofan
  5. Tigeras, pinceles y silicona
  6. Dos paneles solares
  7. Cautin, estaño y pasta térmica
  8. Cables de conexión de componentes
  9. Arduino Uno
  10. Baterias
  11. MIT App Inventor
  12. Visual Studio Code
  13. Simulador de Arduino Uno

Como primeros pasos: Comenzamos diseñando las paredes, el techo y otros elementos estructurales en el cartón. Usa una regla y lápiz para trazar las piezas y recórtalas cuidadosamente con tijeras o un cúter, asegurándonos de que los bordes queden limpios y rectos. Armamos las piezas uniendo los bordes con pegamento resistente, cinta adhesiva o silicón caliente, reforzando las uniones para que la estructura sea sólida y estable. Verifica que la casa encaje bien en la base de tabla triplex y ajusta las dimensiones si es necesario.

Programacion y simuladores: Programamos la placa Arduino Uno en el Arduino IDE para controlar los componentes y verificamos su funcionamiento mediante simulaciones. Luego, desarrollamos una aplicación móvil en MIT App Inventor para interactuar con el sistema Arduino. Finalmente, creamos una página web en Visual Studio Code usando HTML, CSS y JavaScript para complementar el proyecto con información y funcionalidades adicionales.

Últimos detalles y fucionamiento: Finalmente, conectamos la placa Arduino a los paneles solares y a la casa para comprobar el correcto funcionamiento del sistema, asegurándonos de que todos los componentes estén bien integrados y operen según lo esperado. Como último paso, decoramos la casa para mejorar su estética, añadiendo detalles visuales que realcen su presentación final.

Arduino IDE (C/C++): Programamos la placa Arduino Uno para controlar los controladores, definiendo la lógica en C/C++. Este código gestiona la comunicación con otros sistemas (como la app móvil) mediante protocolos como Bluetooth o Wi-Fi, y permite automatizar tareas como encender luces.

MIT App Inventor (Bloques Gráficos): Creamos una app móvil mediante bloques gráficos para interactuar con el Arduino. La app permite enviar comandos (como "ON" o "OFF") y recibir datos en tiempo real a través de Bluetooth o Wi-Fi, facilitando el control y monitoreo desde el teléfono.

HTML (Estructura de la Página Web): Construimos la estructura de la página web con HTML, definiendo los elementos básicos como botones, encabezados y secciones. Es la base estática donde se añade diseño con CSS y funcionalidad con JavaScript.

CSS (Estilo de la Página Web): Aplicamos diseño visual a la página web usando CSS, mejorando su apariencia con colores, fuentes y márgenes. Esto hace la web más atractiva y adaptable a diferentes dispositivos.

JavaScript (Interactividad en la Página Web): Añadimos interactividad a la página web con JavaScript, permitiendo que los elementos reaccionen a eventos, como clics de botones. También usamos este lenguaje para comunicarnos con el Arduino y mostrar datos en tiempo real.

DETALLES DE LA APP

Se diseño una app en MIT App Inventor que controla un LED conectado a un Arduino mediante el módulo Bluetooth HC-06. La app envía comandos para encender ("1") o apagar ("0") el LED, mostrando su estado en tiempo real a través de un indicador. Esto permite al usuario controlar las luces desde su teléfono de manera sencilla.

  • Accesibilidad

    El uso de la app es interactivo y facil de utlizar para el usuario.

  • Descripciones

    El diseño de la app es intuitivo y funcional, incluye botones para encender y apagar el LED. Su interfaz es sencilla y está optimizada para facilitar la conexión y el control del dispositivo mediante Bluetooth.

  • Configuraciones

    La configuracion es sencilla por lo cual cualquier usurio pueda manipular la app.

  • Vinculacion de voz

    Cada vez que se presione un boton sonara la voz del objeto.

  • Eficiencia

    La app es eficiente para comunicarse con el Arduino mediante Bluetooth, lo que minimiza el consumo de recursos. Además, su diseño ligero y su respuesta en tiempo real garantizan una experiencia fluida para el usuario.

  • Descargas

    La app ha estado en constate actulizaciones para la experiencia del usuario
